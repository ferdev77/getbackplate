# src/ (base de aplicacion)

Esta carpeta todavia no contiene implementacion final de Next.js.

Se deja este archivo para fijar la estructura objetivo descrita en:

- `ESTRUCTURA_PROYECTO.md`

Cuando iniciemos el scaffold de Next.js, esta carpeta tendra:

- `app/` (rutas por rol: superadmin, empresa, empleado)
- `modules/` (dominio por modulo)
- `shared/` (UI, utilidades, validaciones, tipos)
- `infrastructure/` (supabase repositorios/adapters)
- `application/` (casos de uso)
- `domain/` (entidades y reglas)
